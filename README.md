# 무너울골: 마지막 산제

고구려 변경 산골 마을에서 끊긴 산제를 둘러싼 조사·의례 서사 RPG 원형입니다.

## 현재 빌드

- 단일 HTML 웹 게임 원형
- 1장: 무너울골 도착
- 2장: 끊긴 제사
- 3장: 금지된 숲
- 4장: 검은등의 자리
- 5장: 마지막 산제
- 종이 질감 그래픽 방향
- 수동 진행 컷신
- 장소 진입형 이벤트
- 단서 기반 선택지

## 실행 방법

브라우저에서 `index.html`을 열면 실행됩니다.

GitHub Pages를 사용할 경우, 저장소 설정에서 Pages 배포 소스를 `main` 브랜치의 root로 지정하면 됩니다.

## 작업 메모

이 저장소는 세계관·서사·게임 원형을 계속 다듬기 위한 프로토타입 저장소입니다.
