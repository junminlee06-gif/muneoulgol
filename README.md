# 무너울골: 마지막 산제

고구려 변경 산골 마을에서 끊긴 산제를 둘러싼 조사·의례 서사 RPG 원형입니다.

## 현재 빌드

- 단일 HTML 웹 게임 원형
- 1장: 무너울골 도착
- 2장: 끊긴 제사
- 3장: 금지된 숲
- 4장: 검은등의 자리
- 5장: 마지막 산제
- 종이 질감 그래픽 방향
- 수동 진행 컷신
- 장소 진입형 이벤트
- 단서 기반 선택지

## 추가 원형

- `index-v2.html`: 선택지-단서-보고서-다음 날 변화 구조를 검증하는 1장 2일 텍스트 원형
- 목적: 보고서를 단순 요약이 아니라 공식 진실 선택 장치로 만들기
- 핵심 흐름: 1일차 조사 → 1일차 보고서 → 2일차 아침 변화 → 2일차 낮 단서 사용 → 2일차 밤 위험 행동 → 2장 조사 명령

## 실행 방법

기존 지도 이동형 원형은 브라우저에서 `index.html`을 열면 실행됩니다.

보고서 루프 구조 검증판은 브라우저에서 `index-v2.html`을 열면 실행됩니다.

GitHub Pages를 사용할 경우, 저장소 설정에서 Pages 배포 소스를 `main` 브랜치의 root로 지정하면 됩니다.

## 설계 문서

- `docs/choice-report-redesign.md`: 선택지-보고서-다음 날 변화 구조 개편안

## 작업 메모

이 저장소는 세계관·서사·게임 원형을 계속 다듬기 위한 프로토타입 저장소입니다.

현재 구조 개편의 우선순위는 기존 `index.html`을 바로 갈아엎는 것이 아니라, `index-v2.html`에서 보고서 루프를 검증한 뒤 지도 이동형 플레이에 통합하는 것입니다.
