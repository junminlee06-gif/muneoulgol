# 마지막 산제 구조 개편안

## 개편 목표

현재 선택지와 보고서는 기록을 남기지만 이후 진행에 미치는 영향이 약하다. 개편판에서는 선택지와 보고서를 다음 날의 상황 변화, 인물 태도, 접근 가능한 장소, 최종 결말 조건에 연결한다.

## 기본 구조

각 장은 2일 또는 3일 분량으로 확장한다.

```text
아침: 전날 보고서 결과 반영
낮: 장소 조사 2~3회
해질녘: 인물 대화 또는 단서 대조
밤: 위험 행동 1회
새벽: 정리 / 꿈 / 목격 / 기록 변화
보고서: 공식 기록 선택
다음 날: 보고서 결과 반영
```

## 선택지의 역할

### 사실 선택
무엇을 사건의 원인으로 볼지 정한다. 단서 해석과 다음 질문을 바꾼다.

### 태도 선택
누구를 믿고 누구를 압박할지 정한다. 인물 신뢰도와 동행 여부를 바꾼다.

### 기록 선택
무엇을 공식 문서에 남길지 정한다. 관아 반응과 마을 여론을 바꾼다.

## 보고서 원칙

보고서는 정답지가 아니라 현실을 바꾸는 장치다.

```text
보고서에 적은 것 = 관아가 공식적으로 믿는 것
보고서에 빠진 것 = 마을이 숨길 수 있는 것
비공식 기록에 남은 것 = 마지막 산제에서만 드러날 수 있는 것
```

## 1차 구현 범위

- 기존 `index.html`은 건드리지 않는다.
- 새 파일 `rework.html`에 개편판 1장만 구현한다.
- 1장 구조는 2일차까지 구현한다.
- 1일차 보고서 선택 결과가 2일차 인물 태도와 사건 흐름에 반영되게 한다.

## 1장 개편 구조

### 1일차 낮
우물, 촌장집, 무당집, 방앗간 중 세 곳을 조사한다.

### 1일차 보고서
다음 중 하나를 공식 기록으로 제출한다.

1. 범 습격 가능성
2. 사람 개입 가능성
3. 산제 관련성
4. 마을 내부 은폐 가능성

### 2일차 아침 변화

- 범 습격: 사량 등장, 막례 신뢰 하락, 사냥 준비
- 사람 개입: 을파 경계 상승, 관아 병졸 등장
- 산제 관련성: 막례가 제문 조각 제공, 관아 신뢰 하락
- 내부 은폐: 월선이 밤에 몰래 접근, 을파 감시 강화

### 2일차 낮
전날 보고서에 따라 새 선택지와 장소 반응이 열린다.

### 2일차 밤
보고서 방향에 따라 다른 밤 행동이 열린다.
